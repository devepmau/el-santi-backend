# el-santi-backend

Backend REST desarrollado con Java y Spring Boot para gestionar usuarios, clientes, servicios y tareas.

## Tecnologías
- Java
- Spring Boot
- Spring Security
- JPA / Hibernate
- PostgreSQL
- Swagger / OpenAPI

## Configuración

Editar el archivo: src/main/resources/application.properties

## Ejecutar el proyecto

Desde Eclipse/IntelliJ ejecutando: BackendApplication.java

## API Documentation

La documentación interactiva de la API está disponible mediante Swagger UI.
Una vez iniciado el backend, acceder a: http://localhost:8080/swagger-ui/index.html
La especificación OpenAPI puede consultarse en: http://localhost:8080/v3/api-docs
